# spinsafe
SpinSafe is an IoT Based Motor Protective & Preventive Controller used to protect motors and machines from serious faults before they appear. 
The repository is been made for DoubleSlash 4.0, IEEE JUSB, Jadavpur University.

- The Repository Contains Hour-to-hour updates & codes regarding the execution of the SpinSafe Model
- You can find a Google Drive Link using which you can find per-hour improvements & modifications.
